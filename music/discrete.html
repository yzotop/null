// концепт

DISCRETE — это музыка, которая звучит только когда число выполняет условие. Не мелодия, не структура — вычисление. Тон появляется, если n простое. Пауза, если составное. Ритм — это распределение простых чисел.

Время здесь не музыкальное. Это математическое время — неравномерное, непредсказуемое, детерминированное. Каждая нота — состояние. Каждая пауза — другое состояние, не менее важное.

Никакой эмоциональной непрерывности. Никакого нарратива. Только итерация по натуральному ряду — и решение для каждого n.

// треки
01 Prime → suno
тон звучит только для простых n. пауза для составных. ритм = теорема о простых числах.
02 Twin Prime → suno
два тона. звучат вместе только если (n, n+2) — оба простые. пауза становится длиннее по мере роста n.
03 Recurrence → suno
каждое следующее значение зависит от предыдущего. рекуррентное соотношение как мелодическая фраза.
04 Divisor Scan → suno
количество делителей n определяет плотность звука. простые числа — минимальная текстура.
