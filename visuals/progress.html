Прогресс-бар — это тоже число. Каждый символ кодирует состояние: заполнено, в процессе, пусто. Шесть вариантов выше показывают, насколько разной может быть эта кодировка при одной и той же шкале от 0 до 100%.

binary — основание 2. noise — стохастика по синусу. matrix — поток с яркой головой. π digits — цифры π напрямую, яркость пропорциональна значению. thermal — экспоненциальный градиент. spectrum — сдвиг hue по hsl.
