NAME

null — личный архив математических объектов, эссе и алгоритмических скетчей.

SYNOPSIS

null { objects | essays | sketches | man }

DESCRIPTION

null — это статичный сайт без базы данных, без аналитики, без подписки. Здесь живут наблюдения за числами и формами, которые я считаю достойными внимания. Каждый объект — отдельный файл с метаданными, эссе и набором маргиналий на полях.

Проект называется null потому, что он начинается с пустоты — с условия, из которого ещё ничего не следует — и постепенно заполняется. Заполнение медленное и не системное.

Любая страница может быть прочитана как самостоятельный текст. Связи между ними — это не оглавление, а сноски: их можно идти, а можно не идти.

BUGS

Эссе пишутся медленнее, чем хотелось бы. Скетчи требуют установленного Sonic Pi, что сужает аудиторию. Шрифт IM Fell English тяжеловат на медленных соединениях, но альтернативы у меня для него нет.

Сообщения об ошибках принимаются по почте, ответы — не гарантированы.
