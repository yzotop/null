NAME

null — картотека математических объектов.

математика как материя. числа существовали до нас.

SYNOPSIS

null [объекты | эссе | визуалы | музыка | карта | find]

DESCRIPTION

На основном сайте я пишу про аналитику — про то как из данных извлекать смысл. Этот проект — про обратное. Про числа у которых смысл уже есть, до всякой аналитики.

Простые числа, π, теорема Гёделя — это не инструменты. Это материал. Как глина или дерево.

null — картотека этого материала. Объекты, эссе, визуалы, музыка.

MUSIC

NULL — алгоритмическая музыка на Suno. Пять EP: DISCRETE · NULL EXECUTION · BIT PARADE · CONSTANTS · QUAD. Треки привязаны к математическим объектам сайта.

DELTA MACHINE — живое кодирование на Sonic Pi. Код как партитура. Математическая структура — единственный композитор. Ключевой альбом: OPERATOR STACK — 8 треков = 8 операторов.
